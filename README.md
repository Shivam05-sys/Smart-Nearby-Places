# Smart-Nearby-Places
Smart Nearby Places Recommender 🌍✨

A modern MERN-stack web application that allows users to discover nearby places based on their mood and preferences. The platform provides detailed information about cafes, temples, colleges, treks, waterfalls, parks, and more with ratings, reviews, maps, and nearby recommendations.

**🚀 Features**
🎯 Mood-based place recommendation
📍 Nearby places using geolocation
🗺 Google Maps & Google Places API integration
⭐ User reviews and ratings system
🌙 Light/Dark theme toggle
🔐 Secure Admin Login
➕ Add / Edit / Delete places dynamically
📷 Place images and descriptions
📱 Fully responsive modern UI

**🛠 Tech Stack**

**Frontend**
React.js
Tailwind CSS
Axios
React Router

**Backend**
Node.js
Express.js
MongoDB
JWT Authentication

**APIs**
Google Maps API
Google Places API

**📌 Use Cases**
Discover nearby cafes, colleges, temples, and tourist spots
View place details, ratings, timings, and maps
Admin can manage all places dynamically
Users can submit reviews and ratings
